# Service Package Menu

Basic PMS Package:
- Equipment inspection
- Cleaning
- Basic calibration check
- Service report
Recommended for low-usage clients.

Standard PMS Package:
- Full inspection
- Calibration
- Minor adjustment
- Parts recommendation
- Service report
Recommended for active equipment users.

Premium Care Package:
- Priority scheduling
- Full PMS and calibration
- Service history dashboard
- Quarterly review
- Discounted parts
Recommended for high-value clients and clients with multiple equipment units.

Fleet Monitoring Add-on:
- GPS tracker integration
- Engine-hour monitoring
- Service due alerts
- Movement and usage dashboard
Recommended for utilities, rental, and field operation clients.
